# Hair Love — фото по продуктам

- `Крем-спрей`: 32 проверенных изображения
- `Термозащита`: 20 проверенных изображений
- `Ламеллярная вода`: 9 проверенных изображений
- `Несколько продуктов`: 11 проверенных групповых и дуэтных изображений
- `Факапы`: 10 дубликатов, спорных или слабых вариантов

В основных папках нет точных файлов-дубликатов.

Папка `Факапы` включает:

- файлы с префиксом `REJECT` — точные дубликаты;
- файлы с префиксом `REVIEW` — почти одинаковые композиции либо кадры,
  требующие дополнительной проверки реалистичности перед публикацией.

Исходные фотографии не удалены и не изменены.
